#Spam Mail Detector using Python

A machine learning project that classifies SMS or email messages as Spam or Not Spam (Ham) using Python and Scikit-learn. The project applies Natural Language Processing (NLP) techniques with TF-IDF Vectorization and trains a Logistic Regression model to identify unwanted messages based on their content.

#Features
Detects spam and non-spam messages
Text preprocessing using TF-IDF Vectorizer
Machine Learning model built with Logistic Regression
Interactive user input for real-time prediction
Simple, beginner-friendly implementation

#Technologies Used
Python
Pandas
NumPy
Scikit-learn
TF-IDF Vectorizer
Logistic Regression
